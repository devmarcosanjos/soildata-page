# SoilData Docs

Este diretório contém a documentação essencial para entender, manter e evoluir o projeto SoilData.

## Estrutura
- `pages.md`: explica como organizar e adicionar novas páginas e rotas.
- `platform.md`: detalha o módulo da plataforma MapBiomas e como manter o shell/layout específicos.
- `libs.md`: descreve bibliotecas e serviços compartilhados, além do fluxo de dados e temas.

Use estes guias quando for adicionar novas funcionalidades, revisar arquitetura ou fazer deploys.
